Từ cơ bản đến chuyên gia — mỗi giai đoạn có kiến thức, công cụ, KPI và dự án thực hành rõ ràng

Giai Đoạn 1⏱ Tuần 1–6 · Cơ bản
Nhập Môn Python – Cú Pháp & Tư Duy Lập Trình
Cài đặt môi trường (VS Code, Anaconda, venv), biến & kiểu dữ liệu, câu lệnh điều kiện, vòng lặp, hàm với tham số & return, list/tuple/dict/set, xử lý chuỗi, import module cơ bản. Học Git và push code lên GitHub từ ngày đầu tiên.
Python 3.xVS Code / PyCharmList / Dict / SetFunction & ScopeGit & GitHubVirtual Environment
🏅 Mục tiêu: CS50P (Harvard) hoàn thành · 4 mini-project trên GitHub · 10 bài LeetCode Easy
Giai Đoạn 2⏱ Tuần 7–12 · Trung cấp thấp
Python Trung Cấp – OOP, File & Công Cụ Chuyên Nghiệp
Lập trình hướng đối tượng (Class, kế thừa, đa hình, dunder methods, @property, @dataclass), xử lý file CSV/JSON/text, Exception Handling chuyên nghiệp, list comprehension nâng cao, decorator, lambda, generator. Thực hành Git workflow với Pull Request.
OOP & Class DesignCSV / JSON / Pathlibtry/except/finallyDecorator & LambdaGit PR WorkflowType Hints & Docstring
🏅 Mục tiêu: Portfolio GitHub đều đặn · Hệ thống quản lý OOP hoàn chỉnh · 20 bài LeetCode Easy
Giai Đoạn 3⏱ Tuần 13–20 · Trung cấp
Thư Viện Thực Dụng – Data, API & Chọn Hướng Chuyên Sâu
NumPy (array, vectorization), Pandas (DataFrame, groupby, merge, pivot, xử lý null), Matplotlib & Seaborn (biểu đồ đẹp), Requests (gọi REST API), BeautifulSoup/Scrapy (web scraping), SQLite & SQLAlchemy cơ bản, Regex, CLI tools với Argparse. Bắt đầu chọn hướng chuyên sâu.
NumPy / PandasMatplotlib / SeabornRequests / REST APIBeautifulSoupSQLite / SQLAlchemyJupyter Notebook
🏅 Mục tiêu: Kaggle Notebook đầu tiên · Phân tích dataset thực tế · Chọn 1 hướng chuyên sâu
Giai Đoạn 4⏱ Tháng 5–9 · Nâng cao
Chuyên Sâu Theo Hướng – Build Project Thực Chiến
[Data/ML] Scikit-learn, TensorFlow/PyTorch, MLflow, feature engineering, model deployment. [Web] FastAPI/Django, PostgreSQL, JWT authentication, Docker, deploy lên cloud. [Automation] Selenium/Playwright, Airflow pipeline scheduling, GitHub Actions CI/CD. [Data Eng] PySpark, dbt, AWS/GCP, Kafka streaming.
FastAPI / DjangoScikit-learn / PyTorchDocker / GitHub ActionsPostgreSQL / RedisApache AirflowPySpark / dbt
🏅 Mục tiêu: Deploy project lên cloud · Pytest coverage ≥80% · Portfolio sẵn sàng xin việc
Giai Đoạn 5⏱ Tháng 10–18 · Chuyên gia
Chuyên Gia & Thực Chiến – Open Source & Sự Nghiệp
Design Patterns (Singleton, Factory, Observer, Strategy), Clean Code & SOLID principles, System Design cơ bản, contribution vào open-source GitHub, publish Python package lên PyPI, mentoring người mới, xây dựng thương hiệu kỹ thuật qua blog/GitHub/YouTube, phỏng vấn Senior Python level.
Design PatternsSOLID / Clean CodeOpen Source PR MergedPyPI PackageSystem DesignTechnical Writing
